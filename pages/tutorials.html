1. Node.js Basics Beginner

Learn the fundamentals of Node.js including modules, npm, and building your first server. Perfect for beginners looking to start backend development with JavaScript.

  • Duration: 2-3 hours of hands-on learning
  • Prerequisites: Basic JavaScript knowledge
  • What you'll learn: Node modules, NPM packages, Express.js basics, Building REST APIs

2. JavaScript Fundamentals Beginner

Master the core concepts of JavaScript including variables, functions, arrays, and objects. Build a strong foundation for modern web development.

  • Duration: 3-4 hours
  • Prerequisites: Basic HTML/CSS knowledge
  • Topics covered: ES6+ features, Async/Await, DOM manipulation, Event handling

3. HTML/CSS Essentials Beginner

Learn to create beautiful, responsive websites with HTML5 and CSS3. Master layouts, animations, and modern web design principles from scratch.

  • Duration: 4-5 hours
  • Prerequisites: None - completely beginner friendly
  • Skills gained: Semantic HTML, Flexbox/Grid, Responsive design, CSS animations

4. Python Programming Intermediate

Dive into Python programming with advanced topics including OOP, file handling, and popular libraries. Great for data science, automation, and backend development.

  • Duration: 5-6 hours
  • Prerequisites: Basic programming knowledge
  • Covered topics: OOP concepts, List comprehensions, Lambda functions, Popular libraries

5. React Basics Intermediate

Build modern, interactive user interfaces with React. Learn components, hooks, state management, and create real-world applications with the most popular frontend library.

  • Duration: 6-8 hours
  • Prerequisites: Strong JavaScript knowledge
  • You'll master: React components, Hooks (useState, useEffect), Props & State, Routing

6. Git & GitHub Essentials Beginner

Master version control with Git and collaborate on GitHub. Learn branching, merging, pull requests, and contribute to open source projects like a pro.

  • Duration: 2-3 hours
  • Prerequisites: Basic command line knowledge
  • Learn: Git commands, Branching strategies, Pull requests, Open source contribution
